  -. Live Sass Compiler
     -> Settings Docs 클릭 
     -> github 사이트에 하단 example 복사
    Example :
        "liveSassCompile.settings.formats":[
            // This is Default.
            {
                "format": "expanded",
                "extensionName": ".css",
                "savePath": null
            },
            // You can add more
            {
                "format": "compressed",
                "extensionName": ".min.css",
                "savePath": "/dist/css"
            },
            // More Complex
            {
                "format": "compressed",
                "extensionName": ".min.css",
                "savePath": "~/../css/"
            }
        ]
     -> vscode의 설정에서 settings:formats에 복붙 후 css경로 변경

        // This is Default.
        {
            "format": "compact",
            "extensionName": ".css",
            "savePath": "~/../css/"
        },
